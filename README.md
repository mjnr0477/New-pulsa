# Pulse Real-time Scaffold

This scaffold contains a React Native (Expo-friendly) structure for a realtime social + marketplace app using Firebase (Firestore, Auth, Storage).

Files included:
- App.js
- src/firebase.js (configured with provided project values)
- src/navigation/*
- src/screens/* (Home, Search, Live, Entertainment feed, Messages, Forums, Marketplace, Profile, Settings, Security, Verification)

To install and run:
1. Ensure you have Node, npm and Expo CLI installed.
2. cd pulse-realtime-scaffold
3. npm install
4. expo start

Enable Firebase in your console (Auth, Firestore, Storage) and adjust security rules.
